# LinuxCLI
Linux CLI on Windows with C++17

```
cd LinuxCLI
g++ main.cpp commands/* -o CLIProgram
CLIProgram.exe
```

### Implemented

- [x] ls
- [x] pwd
- [x] cd
- [ ] mkdir
- [ ] mv
- [ ] cp
- [ ] rm
- [x] echo
- [x] clear
- [x] exit (software exit command)
- [x] info (information about the software)
