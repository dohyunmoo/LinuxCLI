# LinuxCLI
Mock Linux CLI with some file management commands on Windows with C++17

```
cd LinuxCLI
g++ main.cpp commands/* -o CLIProgram
CLIProgram.exe
```

### Implemented

- [x] ls
- [x] pwd
- [x] cd
- [x] mkdir
- [x] cp
- [x] rm
- [x] echo
- [x] clear
- [x] exit (software exit command)
- [x] info (information about the software)
